ThreeDimensionalMesh

Bases: Delaunay

An abstraction of a 3D mesh that extends the functionality of Numpy's Delaunay class. Some notable differences are that this implementation is assuming a "2.5D" interpretation of the mesh which triangulates via the x and y coordinates then projects into 3D space.
